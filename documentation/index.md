# Autonomous Workflow — System Documentation

Welcome to the official documentation for the **Autonomous Workflow** project. This system is a sophisticated multi-agent ecosystem designed for deep research, trend analysis, and professional reporting.

## 🤖 The Agent Ecosystem

The project consists of four specialized agents that collaborate in a linear data pipeline:

1.  **[Researcher Agent](researcher_agent.md)**: Scours the web for the latest data, articles, and news regarding a specific query.
2.  **[Analyst Agent](analyst_agent.md)**: Processes raw data into high-level insights using NLP, Sentiment Analysis, and Topic Clustering.
3.  **[Forecaster Agent](forecaster_agent.md)**: Analyzes temporal data to project future trends and seasonal patterns.
4.  **[Writer Agent](writer_agent.md)**: Synthesizes all findings into a premium, human-readable report.
5.  **[Web Interface](web_interface.md)**: Unified FastAPI dashboard for real-time orchestration and monitoring.
